Project description for RDF::RDFa.

RDF.rb plugin for parsing/serializing RDF/XML, Turtle and RDFa data using the Raptor RDF Parser library.

Implemented Specifications
  • RDFa in XHTML (RDFa Core 1.0)
  • RDFa Core 1.1
  • XHTML+RDFa 1.1
  • HTML+RDFa 1.1
  • RDFa Lite 1.1
  • Role Attribute
